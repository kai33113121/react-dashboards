# Dashboards — React + Tailwind CSS

Colección de 3 paneles administrativos construidos con React, Tailwind CSS y Recharts.

## Proyectos

### 1. `admin-dashboard` — Panel Administrativo
Panel de control empresarial con tema oscuro (verde/negro).
- Sidebar de navegación con estado activo
- Tarjetas de métricas con tendencias
- Gráfica de área (Ingresos vs Gastos)
- Feed de actividad en tiempo real
- Tabla de pedidos con badges de estado

### 2. `analytics-dashboard` — Analítica Web
Dashboard de tráfico web con tema claro/minimalista.
- KPIs de visitas, usuarios únicos y rebote
- Gráfica de líneas por fuente de tráfico
- Desglose de dispositivos con PieChart
- Tabla de páginas más visitadas

### 3. `finance-dashboard` — Finanzas Personales
Tracker de gastos personales con tema oscuro (violeta/rosa).
- Balance card con gradiente
- Barras de presupuesto por categoría
- Gráfica de barras Ingresos vs Gastos
- Lista de movimientos recientes

## Stack común
- React 18
- Tailwind CSS 3
- Recharts
- Lucide React
- Vite

## Instalación (por proyecto)

```bash
cd <nombre-del-dashboard>
npm install
npm run dev
```
