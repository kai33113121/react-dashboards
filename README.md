React Dashboards

Colección de 3 paneles administrativos construidos con React, Tailwind CSS y Recharts.

Proyectos

Admin Panel
Panel de control empresarial con tema oscuro.
- Sidebar de navegación con estado activo
- Tarjetas de métricas con indicadores de tendencia
- Gráfica de área — Ingresos vs Gastos
- Feed de actividad en tiempo real
- Tabla de pedidos con badges de estado por color

Analytics Dashboard
Dashboard de tráfico web con tema claro y minimalista.
- KPIs de visitas, usuarios únicos, tiempo promedio y rebote
- Gráfica de líneas por fuente de tráfico (orgánico, directo, referido)
- Desglose de dispositivos con PieChart
- Tabla de páginas más visitadas con métricas

Finance Dashboard
Tracker de finanzas personales con tema oscuro violeta.
- Tarjeta de balance con gradiente animado
- Barras de presupuesto por categoría con alerta de sobrepaso
- Gráfica de barras Ingresos vs Gastos mensual
- Lista de movimientos recientes con iconos por categoría

Stack

| Tecnología   | Uso                         |
|--------------|-----------------------------|
| React 18     | Arquitectura de componentes |
| Tailwind CSS | Estilos utility-first       |
| Recharts     | Gráficas y visualizaciones  |
| Lucide React | Iconos como componentes     |
| Vite         | Bundler y dev server        |

Instalación

Cada dashboard es independiente. Entra a la carpeta y corre:

```bash
cd admin-dashboard
npm install
npm run dev
```
